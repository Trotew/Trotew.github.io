E. 简单结论+暴力

题意

有$n$只队伍，每个队伍有一个封榜前的得分$a_i$，有一个封榜后的得分$d_i$，得分越大越靠前，如果有多支队伍得分相同，队伍$id$小的靠前。

你可以决定展示封榜后结果的队伍顺序，如果展示前一支队伍的排名为$x_i$，展示后这支队伍的排名为$y_i$，那么观众将鼓掌$|x_i-y_i|$的时间。

问最大的鼓掌时间。

$n,a_i\le 100$

$-100\le d_i\le 100$

题解

以下两个结论请尝试认真思考并加以证明。

引理1：两支队伍$u,v$，当且仅当如果先展示$v$，则$u$无法触及，如果先展示$u$，则$u$可以超越$v$；再展示$v$，$v$又可以超越$u$时。$u$必须在$v$之前展示。其余时候谁先谁后并不影响答案。

引理2：上述方式产生的$u\Rightarrow v$连有向边后，图必然是一张有向图。

由上，暴力连边后拓扑排序求答案即可。

时间复杂度$O(n^3)$

I. 贪心

题意

现在有$n$件物品，每件物品可以放进$A$中获得$a_i$，也可以放进$B$中获得$b_i$。

现在$A$中可以容纳$a$件物品，$B$中可以容纳$b$件物品，问放进$A,B$中的物品的获得利益最大是多少，并输出方案。

$n,a,b\le 3000$

题解

结论：可以先贪心装满$A$，然后考虑装满$B$。

  • 选出一件没有装入$A$的物品$i$，并且装入$B$，获得$b_i$。
  • 选出一件已经装入$A$的物品$j$，并且装入$B$，并选出一件没有装入$A$的物品$i$装入$A$，获得$b_j-a_j+a_i$。

两边取最大即可。

上述结论可以利用网络流证明（本质是优化网络流，直接跑网络流也可以AC）。

时间复杂度$O(n\log n)$

当然如果想DP也可以，但是很麻烦。
